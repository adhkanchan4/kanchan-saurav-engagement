KANCHAN & SAURAV ENGAGEMENT WEBSITE

UPLOAD TO GITHUB
1. Open this folder.
2. Select every file inside it.
3. Upload all files together to the root of your GitHub repository.
4. Do not upload the outer folder by itself.

NETLIFY
Connect the GitHub repository to Netlify. No build command is needed. The publish directory is the repository root.

RSVP
Responses appear in Netlify under Forms > engagement-rsvp.
