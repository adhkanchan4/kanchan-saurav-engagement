KANCHAN & SAURAV ENGAGEMENT WEBSITE

UPLOAD TO NETLIFY
1. Unzip this folder.
2. Drag the entire folder into Netlify Drop, or connect the folder through GitHub.
3. Netlify automatically detects the RSVP form after the first deployment.
4. View submissions in Netlify: Site dashboard > Forms > engagement-rsvp.

UPLOAD TO GITHUB
1. Create a new GitHub repository.
2. Upload every file and the assets folder.
3. Connect the repository to Netlify.
4. Build command: leave blank.
5. Publish directory: .

RSVP RESPONSES
The form is configured for Netlify Forms. It includes:
- Full name
- Attendance response
- Optional message

To send Netlify submissions into Google Sheets, use Netlify's Zapier integration or Make.com:
Trigger: New Netlify form submission
Action: Add row in Google Sheets
