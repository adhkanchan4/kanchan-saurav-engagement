KANCHAN & SAURAV ENGAGEMENT WEBSITE

Everything is in ONE folder for easy GitHub and Netlify upload.
Do not separate or rename the image files.

UPLOAD THESE FILES TOGETHER:
- index.html
- styles.css
- script.js
- netlify.toml
- google-apps-script.js
- SETUP-GOOGLE-SHEETS.txt
- invitation-pearl-frame.png
- pearl-floral-background.png
- ivory-envelope-red-ks-seal.png
- engagement-photo.jpeg

NETLIFY
1. Upload this entire folder to GitHub, then connect the repository to Netlify.
2. Or drag the whole folder directly into Netlify Drop.
3. RSVP responses appear in Netlify under Forms > engagement-rsvp.

GOOGLE SHEETS
Follow SETUP-GOOGLE-SHEETS.txt, then paste the deployed Apps Script URL
into GOOGLE_SCRIPT_URL at the top of script.js.
