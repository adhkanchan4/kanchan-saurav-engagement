GOOGLE SHEETS HEADCOUNT + EMAIL NOTIFICATIONS

1. Create a blank Google Sheet named:
   Kanchan & Saurav RSVPs

2. In the Sheet, click:
   Extensions → Apps Script

3. Delete any existing code.

4. Open:
   google-apps-script.gs

5. Copy all code into Apps Script and save.

6. Select:
   setupGuestList

7. Click Run and approve permissions.

8. Click:
   Deploy → New deployment → Web app

9. Set:
   Execute as: Me
   Who has access: Anyone

10. Click Deploy and copy the Web App URL.

11. Open index.html in a plain-text editor.

12. Find:
    const GOOGLE_SCRIPT_URL = "PASTE_YOUR_GOOGLE_APPS_SCRIPT_URL_HERE";

13. Replace the placeholder with your Web App URL.

14. Save index.html and upload it to GitHub.

RESULTS

- Every RSVP is emailed to adhkanchan4@gmail.com
- RSVP Responses stores every answer
- Guest List shows Accepted, Declined, or Pending
- Headcount Summary totals all three categories
- Netlify Forms keeps a backup

IMPORTANT

Add all invited guests to Column A in Guest List.
Set Column B to Pending before sending invitations.
