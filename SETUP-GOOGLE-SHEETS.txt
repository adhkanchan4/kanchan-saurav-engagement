GOOGLE SHEETS RSVP SETUP — DO THIS ONCE

This website is complete. The only missing piece is connecting the RSVP form to your Google Sheet.

1. Open Google Sheets and create a blank spreadsheet.
   Name it: Kanchan & Saurav RSVPs

2. In the spreadsheet, click:
   Extensions → Apps Script

3. Delete any code you see.

4. Open the file named:
   google-apps-script.gs

5. Copy all of that code and paste it into Apps Script.

6. Click Save.

7. At the top, choose the function:
   setupGuestList
   Then click Run.

8. Google will ask for permission.
   Approve it.

9. In Apps Script, click:
   Deploy → New deployment

10. Click the gear beside "Select type" and choose:
    Web app

11. Set:
    Execute as: Me
    Who has access: Anyone

12. Click Deploy.

13. Copy the Web App URL.

14. Open index.html in a text editor.

15. Find this line:
    const GOOGLE_SCRIPT_URL = "PASTE_YOUR_GOOGLE_APPS_SCRIPT_URL_HERE";

16. Replace only the words inside the quotation marks with your Web App URL.

17. Save index.html.

18. Upload these files to your GitHub repository:
    - index.html
    - invitation.png
    - google-apps-script.gs
    - SETUP-GOOGLE-SHEETS.txt

19. GitHub will update Netlify automatically.

HOW TO TRACK EVERYONE

In your Google Sheet:

- RSVP Responses:
  Every submitted RSVP appears here.

- Guest List:
  Put every invited guest's exact name in Column A.
  Column B will automatically change to:
  Accepted
  Declined
  Pending

Anyone still marked Pending has not replied.
