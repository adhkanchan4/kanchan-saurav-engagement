GOOGLE SHEETS HEADCOUNT + EMAIL NOTIFICATIONS

1. Create a blank Google Sheet named:
   Kanchan & Saurav RSVPs

2. In the Google Sheet, click:
   Extensions → Apps Script

3. Delete the code already shown in Apps Script.

4. Open the downloaded file:
   google-apps-script.gs

5. Copy all the code from that file and paste it into Apps Script.

6. Click Save.

7. At the top, select the function:
   setupGuestList

8. Click Run and approve the requested Google permissions.

9. Click:
   Deploy → New deployment

10. Choose:
    Type: Web app
    Execute as: Me
    Who has access: Anyone

11. Click Deploy.

12. Copy the Web App URL.

13. Open index.html in a plain-text editor.

14. Find:
    const GOOGLE_SCRIPT_URL = "PASTE_YOUR_GOOGLE_APPS_SCRIPT_URL_HERE";

15. Replace only the placeholder with your Web App URL.

16. Save index.html and upload all six files to GitHub.
